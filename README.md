# GOTFlow: Learning Directed Population Flow with Generalised Optimal Transport

Official repository for **GOTFlow (Generalised Optimal Transport Flow)**.

## Overview

<p align="center">
  <img src="Figures/Figure_1.png" width="40%" />
</p>

*Figure: Overview of the GOTFlow concept. For each dataset, samples are represented in their original feature space and assigned to ordered target states. GOTFlow uses Optimal Transport to estimate a transport plan between states. The resulting transport plan is then visualised and analysed to characterise state-to-state transitions, including drift and feature-level attribution analyses that highlight which variables change most strongly along the trajectory.*


This repo provides:
- a **toy example** to illustrate the GOTFlow pipeline end-to-end, and  
- a **TCGA-BRCA experiment** to reproduce the main figures and supplementary plots from the paper.
